Mattermost end user documentation
=================================

The following resources will help you get the most out of Mattermost Channels, Mattermost Playbooks, and Mattermost Boards:

.. toctree::
   :maxdepth: 1
   :hidden:

   Mattermost Channels </guides/channels>
   Mattermost Playbooks </guides/playbooks>

* :doc:`Mattermost Channels </guides/channels>` - Learn about 1:1 and group messaging, integrated voice/video conferencing, file, image, and link sharing, rich markdown formatting, and a fully searchable message history.
* :doc:`Mattermost Playbooks </guides/playbooks>` - Learn about manage pre-built, configurable checklists that define a repeatable process for teams to achieve specific and predictable outcomes.
* :doc:`Mattermost Boards </guides/boards>` - Learn how to manage projects and tasks via a familiar Kanban board structure
